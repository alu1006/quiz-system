🦫 什麼是 Bebras？

Bebras（國際運算思維挑戰）是一項源自立陶宛的國際性資訊科學競賽，每年有超過 80 個國家、數百萬學生參與。 Bebras 不要求任何程式設計經驗，而是透過有趣的情境題目，考驗學生的邏輯推理、問題分解、模式辨識與演算法思維能力。

運算思維是什麼？

一套不需要寫程式就能學會的解題思考法。點擊看看每個能力：

🔍 問題拆解 Decomposition

把一個大問題拆成幾個小問題，各個擊破。
例如：整理房間 → 先整理桌面、再整理衣櫃、最後拖地。

📐 模式辨識 Pattern Recognition

觀察問題之間的共同點，找出規律。
例如：1, 3, 5, 7, ? → 發現都是 +2，答案是 9。

🎯 抽象化 Abstraction

去掉不重要的細節，只留下關鍵資訊。
例如：地鐵路線圖不畫出真實地形，只呈現站與站之間的連接關係。

⚙️ 演算法設計 Algorithm Design

把解法寫成一步一步的明確指令。
例如：泡麵的步驟 → 1.煮水 2.打開調味包 3.放入麵條 4.等3分鐘。
